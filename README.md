# emergency_system

Un projecto hecho en flutter, en busca de una solución.

## Motivación

Luego de una situación comprometedora en la Universidad Nacional 
de Río Negro, nace esta idea para respaldar a las personas
que sufran de una situación similar, pudiendo aumentar
las posibilidades de salvación y ayuda.

## ¿Cómo usar? 

De manera simple, se instala la aplicación, se rellenan
los datos solicitados, y muy importante permitir las 
notificaciones de la aplicación, ya que esto permite 
acceder a los datos sin la necesidad de desbloquear el celular,
esta es una de las funcionalidades fundamentales de la aplicación,
ya que un tercero puede alterar estos datos o usar esa
información con malas intenciones.

## Detalles a tener en cuenta

Es muy importante utilizar con responsabilidad y consciencia.
La vida del usuario puede estar en riesgo y esta aplicación
es capaz de hacer la diferencia.
